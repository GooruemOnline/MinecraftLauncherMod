## mod
이 디렉토리는 `/GooruemLauncher/resources/mod.zip`의 내용물을 포함합니다.   
`mod.zip`이 GitHub LFS의 할당량 60% 이상 (110% 이상 예정)을 차지하는 관계로, 내부 파일을 하나하나 찢어서 넣는 게 맞다고 판단하였습니다.

Open Source Lisence::
각각의 모드는 그 모드 내부에 라이센스가 포함되어 있습니다.   
FancyMenu의 설정은 `전도현`이 구름 온라인 전용으로 제작, 작업하고 있습니다.

## 라이브러리 업데이트 안내
1.16.5의 `RPG` 서버, 1.18.2의 `반야생` 서버의 런처를 분리하기 위해, `mod` 디렉토리를 아얘 [`wild`](wild), [`adventure`](adventure)로 쪼갰습니다.   
1.18.2의 부팅 테스트는 완료되었습니다. 1.16.5는 테스트를 준비하고 있습니다.
